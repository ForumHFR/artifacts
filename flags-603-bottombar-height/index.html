Hauteur de la barre du bas sans libellés

Quand les libellés sont masqués, la barre garde une hauteur de barre avec libellés. Règle voulue : reprendre le gap icône → haut de barre de la barre avec libellés, et l'appliquer en haut ET en bas de l'icône. Ajuste le padding ci-contre pour fixer la bonne hauteur ; je code la valeur retenue dans IconOnlyBottomBar.

Référence — AVEC libellés80 dp (M3 standard)

SANS libellés — proposé— dp

padding haut12 dp
icône24 dp
padding bas12 dp
hauteur barre48 dp

La référence avec libellés place le groupe icône+libellé centré dans 80 dp. Sans libellé, l'icône doit être centrée avec un padding symétrique — c'est ce padding qui fixe la hauteur. La zone système s'ajoute en plus (identique sur les deux barres).
